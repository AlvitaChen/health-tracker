# Health Tracker

A personal health tracking app with food logging, menstrual cycle tracking, body composition, and nutrition progress.

## Features
- AI-powered food logging via photo or text
- Menstrual cycle phase tracking with auto-suggestions
- Daily steps and activity notes
- Body composition tracking over time
- Progress views: daily, weekly, monthly, and by phase
- Personalized nutrition goals
- Dark mode support

## Tech Stack
- React (via CDN)
- Supabase (database)
- Vercel (hosting)
- Anthropic Claude API (food estimation)

## Setup
1. Clone this repo
2. Update `SUPABASE_URL` and `SUPABASE_ANON_KEY` in `app.js`
3. Deploy to Vercel
